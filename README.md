# Gen AI Floor Plan - Backend

## 📌 Project Overview
This repository contains the backend API for the Gen AI Floor Plan application. The backend is built using FastAPI and provides AI-powered floor plan generation and related services.

## 🚀 Live Demo

### Frontend
https://gen-ai-floor-plan.vercel.app/

### Backend API
https://genaifloorplanbackend-production.up.railway.app/

## 🛠️ Tech Stack
- Python
- FastAPI
- MongoDB Atlas
- Groq API
- Railway
- REST API

## 📂 Frontend Repository
https://github.com/Tesigit/GenAIFloorPlan

## ⚙️ Run Locally

1. Clone the repository
```bash
git clone https://github.com/Tesigit/GenAIFloorPlanBackend.git
```

2. Install dependencies
```bash
pip install -r requirements.txt
```

3. Create a `.env` file and add the required environment variables.

4. Start the server
```bash
uvicorn main:app --reload
```
